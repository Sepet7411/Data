# Customer Churn Analysis Dashboard
![Dashboard Screenshot](Customer%20Churn%20Analysis./Знімок%20екрана%202026-05-02%20165423.png)
## Project Overview
This project features an interactive Customer Churn Analysis Dashboard built in Microsoft Excel. The primary objective is to visualize key retention metrics, identify the root causes of customer attrition, and analyze demographic and behavioral patterns to support data-driven decision-making.
## Key Performance Indicators (KPIs)
The dashboard tracks three critical high-level metrics:
* **Total Customers:** 6,687
* **Churned Customers:** 1,796
* **Churn Rate:** 26.86%
## Dashboard Features & Visualizations
The analysis is categorized into several strategic dimensions:
1. **Churn Reasons:** A ranked bar chart identifying why customers leave. The data shows a strong impact from competitors (better offers, superior devices, and higher internet speeds).
2. **Competitor Analysis:** A donut chart detailing competitor advantages, where "Better Devices" and "Better Offers" each account for 37% of the churned base.
3. **Demographics Profile:** Analyzes churn by age groups. Notably, the "Under 30" segment represents 44% of the total churn.
4. **Age Group Trend:** A combination chart showing the relationship between total customer volume and churn probability across various age brackets (19 to 88 years).
5. **Data Consumption Analysis:** A stacked bar chart visualizing how data usage (e.g., <5GB vs >10GB) correlates with customer loyalty.
6. **Geographic Heatmap:** A conditional formatting table showing the Churn Rate by state. For instance, CA (California) shows a critical churn rate of 75%.
## Technical Implementation
Data Aggregation: Utilized Pivot Tables to summarize large datasets into meaningful business insights.
Interactivity: Implemented Slicers (e.g., International Plan filter) to allow users to drill down into specific segments.
Data Visualization: Applied advanced Excel charting techniques, including combo charts and donut charts, with a clean and professional UI/UX design.
Conditional Formatting: Used heatmaps to highlight high-risk geographic areas instantly.
## Use Cases
* **Retention Strategy:** Identifying high-risk segments (like the Under 30 group) for targeted marketing campaigns.
* **Competitive Intelligence:** Understanding specific areas where competitors are outperforming the company.
* **Operational Optimization:** Analyzing service-related churn reasons to improve technical support or product offerings.
This project demonstrates proficiency in data cleaning, exploratory data analysis (EDA), and business intelligence reporting.

# Аналіз відтоку клієнтів (Customer Churn Analysis Dashboard)
## Опис проекту
Цей проект представляє собою інтерактивний дашборд, створений в Microsoft Excel, для аналізу відтоку клієнтів (Churn Rate). Мета інструменту — візуалізувати ключові метрики, виявити основні причини відходу клієнтів та проаналізувати демографічні та поведінкові фактори, що впливають на лояльність користувачів.
## Основні KPI (Ключові показники)
На дашборді виведено три головні метрики, що дають миттєве розуміння ситуації:
* **Total Customers (Загальна кількість клієнтів):** 6 687
* **Churned Customers (Кількість клієнтів, що пішли):** 1 796
* **Churn Rate % (Відсоток відтоку):** 26,86%
## Структура та візуалізація
Дашборд розділений на логічні блоки для глибокого аналізу даних:
1.  **Churn Reasons (Причини відтоку):** Гістограма, що ранжує фактори відтоку. Найбільший вплив мають дії конкурентів (кращі пропозиції, сучасніші пристрої, вища швидкість інтернету).
2.  **Competitor Churn Analysis (Аналіз відтоку до конкурентів):** Пончик-діаграма, яка деталізує причини переходу до конкурентів (по 37% припадає на кращі пристрої та вигідніші пропозиції).
3.  **Churn by Demographics (Відтік за демографією):** Розподіл клієнтів, що пішли, за віковими категоріями. Найбільша частка відтоку (44%) припадає на категорію "Under 30".
4.  **Age Group Analysis (Аналіз за віковими групами):** Комбінована діаграма, що показує обсяг клієнтської бази та динаміку відтоку в різних вікових діапазонах (від 19 до 88 років).
5.  **Consumption Churn (Відтік за обсягом споживання):** Стовпчаста діаграма, що аналізує лояльність залежно від обсягу використаного інтернет-трафіку.
6.  **Geographic Churn Rate (Географічний аналіз):** Таблиця з умовним форматуванням (Heatmap), що демонструє рівень відтоку за штатами (наприклад, CA має критичний показник 75%).
## Використані інструменти та методи
* **Microsoft Excel:** Побудова зведених таблиць (Pivot Tables) для агрегації даних.
* **Аналіз даних:** Розрахунок мір відтоку та категорізація причин.
* **Data Visualization:** Використання слайсерів (Slicers) для інтерактивної фільтрації (на скріншоті вибрано фільтр `Intl Plan: yes`) та умовного форматування для виявлення аномалій.
